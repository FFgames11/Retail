# Stage Template

This folder stores the current approved board state as a reusable stage template.

Included snapshot:
- `index.html`
- `style.css`
- `script.js`

Template contents:
- portrait mobile-first board shell
- centered `5 x 11` playable path inside a `7 x 13` board frame
- placeholder asset overlay
- player circle and click glow layer
- current click redirect rules
- current collider and walkable tile rules
- current placeholder labels and player label

Use this folder as the base when creating future stage variations that move assets around without rebuilding the board system from scratch.
